# TreeHole 前端

本项目为 TreeHole 树洞网站的前端，基于 Vue 3 + Vite + Element Plus。

## 功能简介
- 留言展示、发布、点赞、踩、回复
- 与 FastAPI 后端接口对接

## 启动方式

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 相关技术
- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Element Plus](https://element-plus.org/)
